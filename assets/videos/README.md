# Creative Approach - Video Assets

## Required Videos

### Main Videos
- **drone-footage.mp4** - Hero background video
  - Recommended: 1920x1080 (Full HD)
  - Duration: 10-30 seconds (will loop)
  - Format: MP4 (H.264 codec)
  - Compressed for web

- **showreel.mp4** - Company showreel/demo video
  - Recommended: 1920x1080 (Full HD)
  - Duration: 2-5 minutes
  - Format: MP4 (H.264 codec)
  - Compressed for web

## Video Optimization

1. **Compress videos** for web delivery:
   - Use HandBrake (free)
   - Target bitrate: 5-8 Mbps for 1080p
   - Use H.264 codec for maximum compatibility

2. **Multiple formats** (optional for better compatibility):
   - MP4 (H.264) - Primary format
   - WebM - For modern browsers
   - OGV - Fallback for older browsers

3. **Lazy loading**: Videos are set to load efficiently with proper attributes:
   - `autoplay muted loop playsinline` for background videos
   - `controls` for modal videos
