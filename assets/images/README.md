## Image Optimization Tips

1. **Compress all images** using tools like:
   - TinyPNG (https://tinypng.com)
   - ImageOptim
   - Squoosh (https://squoosh.app)

2. **Use appropriate formats**:
   - JPEG for photos
   - PNG for logos and graphics with transparency
   - WebP for modern browser optimization

3. **Naming convention**:
   - Use lowercase
   - Use hyphens instead of spaces
   - Be descriptive

4. **Alt text**: All images in the HTML have descriptive alt attributes for accessibility and SEO
